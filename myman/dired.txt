■困ったときのURL
http://www.bookshelf.jp/texi/emacs-20.6-man-jp/emacs_29.html

■起動
C-x d

■コマンド
移動
  n , SPC : 次の行へ
  p : 前の行へ
ファイル操作
  RET, f : 現在行のファイルを開く
  D : 指定したファイルを削除する※
  R : 指定したファイル名を変更する※
  C : 指定したファイルをコピーする※
  x : マークしたファイルを削除する
  + : ディレクトリを作成する
マーク操作
  d : 削除候補としてマークする
  m : マークする
  *% : 正規表現でマークする
  <Backspace> : 一行上のマークを外す
  u : 現在行のマークを外す
  *! : マークを全て外す
基本操作
  C-_ : Undo
  q : ウィンドウを閉じる

※が付いているのは、「マークがあればマークしたファイル、マークがなければ現在行のファイルに対して実行」
