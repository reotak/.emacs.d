\item 表は|とTAB
\begin{itemize}
\item 行頭を|で始める
\item 最初の行を書いたらTAB
\item 以降一つセルを書くごとにTAB
\item 枠線を引くときは |- TAB
\item 表計算をするときには C-c \} をすると番号がわかる
\end{itemize}
\item 翻訳後にそのまま残したい文章
\begin{itemize}
\item 一行ならば : で始める
\item 複数行は \#+BEGIN$_{\mathrm{EXAMPLE}}$   \#-END$_{\mathrm{EXAMPLE}}$
\end{itemize}
\item[/], [], の後にC-c C-c でチェックリスト

\end{itemize}
